## Overview
Developer Studio is an IDE for everyone. It features a clean and simple UI and lot's of features!

## Supported Languages
- [[PaneScript]]
- C++
- Java
- Python
- Javascript (NodeJS backend)
- HTML & CSS
- PHP
- Rust
- Fortran
- Ruby
- Lua
- C#
- Kotlin
- XML
- YML

Supports JSON, Postgres, MongoDB and SQLite as database backends out of the box.
